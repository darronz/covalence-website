**Block 1 — sr-only utility class (per D-09):**
```css
/* === Accessibility === */
.sr-only {
  position: absolute;
  width: 1px;
  height: 1px;
  padding: 0;
  margin: -1px;
  overflow: hidden;
  clip: rect(0, 0, 0, 0);
  white-space: nowrap;
  border-width: 0;
}
```

**Block 2 — :focus-visible styles for all interactive elements (per D-03, D-04, D-05):**
```css
a:focus-visible,
button:focus-visible,
input:focus-visible,
select:focus-visible,
textarea:focus-visible,
[tabindex]:focus-visible {
  outline: 2px solid var(--accent-subtle);
  outline-offset: 2px;
}
```

Do NOT use `:focus` (per D-04). Do NOT use `box-shadow` (per D-04). The `:focus-visible` pseudo-class only activates on keyboard/programmatic focus, not mouse clicks.

The `var(--accent-subtle)` resolves to `#a78bfa` — a purple that is visible on the dark `--bg-primary` (#08060d) background.
