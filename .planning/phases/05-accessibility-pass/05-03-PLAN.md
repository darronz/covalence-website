**Step 3: Run axe-core scan (per D-01, D-02).**

Serve the dist/ directory and scan both pages:

```bash
npx serve dist -l 4173 &
SERVER_PID=$!
sleep 2

# Scan home page
echo "=== Scanning / ==="
npx axe http://localhost:4173/ --tags wcag2a,wcag2aa 2>&1

# Scan releases page
echo "=== Scanning /releases/ ==="
npx axe http://localhost:4173/releases/ --tags wcag2a,wcag2aa 2>&1

kill $SERVER_PID
```
