### Step 3 — SC3 / SEO-02 — sitemap coverage

```bash
# Both sitemap files exist
test -f dist/sitemap-index.xml
test -f dist/sitemap-0.xml

# sitemap-index references exactly one inner sitemap
[ "$(grep -c '<loc>' dist/sitemap-index.xml)" = "1" ]

https://covalence.app/docs/spaces/
https://covalence.app/docs/under-the-hood/

for url in $EXPECTED_URLS; do

# Total URL count ≥ 11
```

### Step 4 — SC4 / SEO-03 — canonical on every public page

```bash
PAGES="dist/index.html dist/releases/index.html dist/posts/index.html dist/docs/index.html dist/docs/under-the-hood/index.html dist/docs/getting-started/index.html dist/docs/spaces/index.html dist/docs/core-memories/index.html dist/docs/mcp-tools/index.html dist/docs/ai-instruction/index.html dist/docs/keyboard-shortcuts/index.html"
```
