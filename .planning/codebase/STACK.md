**Core:**

Provides the `/docs/` route, docs content collection loader (`docsLoader`), docs schema (`docsSchema`), UI components (e.g. `Tabs`, `TabItem`), and the bundled Pagefind search index.

## Key Dependencies

**Critical (runtime `dependencies`):**
- `astro` `^6.1.6` — SSG framework. Resolved to a version that declares `sharp ^0.34.0` as an optional dependency for image processing.
- `@astrojs/starlight` `^0.38.3` — Docs framework. Pulls in `pagefind ^1.3.0` (static search) and other internals transitively.

**Dev (`devDependencies`):**
- `@fontsource/inter` `^5.2.8` — Inter font files. The repo does not import the fontsource package at runtime; instead `src/styles/global.css` declares `@font-face` rules that point to self-hosted woff2 files in `public/fonts/` (`inter-latin-400.woff2`, `inter-latin-600.woff2`, `inter-latin-700.woff2`). The npm dep appears to be the source-of-truth for those binary files.
- `@fontsource/jetbrains-mono` `^5.2.8` — JetBrains Mono font files, served as `public/fonts/jetbrains-mono-latin-400.woff2`.

**Infrastructure:**
- `pagefind` `^1.3.0` (transitive via Starlight) — Static full-text search, built into `dist/pagefind/` at `astro build` time.
- `sharp` `^0.34.0` (optional, transitive via Astro) — Image optimisation pipeline.

## Platform Requirements

**Development:**
- Node.js 22.16.0 (per `.nvmrc`)
- npm (per lockfile)
